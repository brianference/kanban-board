# FlowBoard v5.0.0 — Major release

**Release date:** 2026-07-20  
**Codename:** Comments, polish & navigation  
**Production:** https://kanban-board-public.pages.dev

## Summary

v5 builds on the Cloudflare Pages + D1 foundation (v4) with comment images and @mentions, client image compression, full test campaign tooling, marketing README, and a wave of UX navigation fixes so primary actions always land where users expect.

## Highlights

### Product
- **Comment images** — up to 4 images per comment (D1-backed, auth-served)
- **@mentions & #tags** — autocomplete from project members, highlighted body, in-app notifications for mentions
- **Auto image compress** — large screenshots resized/compressed in-browser before upload
- **Save & return to board** — task edits (priority, title, etc.) save then navigate back to the board
- **Back to board / Cancel** on task detail; login returns to the protected URL you tried to open
- **Quick-add task** on the board (stays on board after create)
- **Search** breadcrumb + back to board when scoped by project
- **Black footer**, unified workspace chrome, solid priority pills (v4 design system refined)

### Quality & ops
- **Test campaign** suite: API smoke/security, Playwright visual/UX/mobile, 100-user sim
- **100 UAT cases** catalog + bugs/learnings report
- **CSP fix** — theme bootstrap via external `theme-boot.js`
- Marketing README with production screenshots
