# FlowBoard v6.0.0 — Major release

**Release date:** 2026-07-20  
**Codename:** Production chrome lock  
**Production:** https://kanban-board-public.pages.dev

## Summary

v6 locks the full multi-user product as a stable major: Cloudflare Pages + D1 edge stack, accounts and JWT auth, drag-and-drop boards, search, task images with auto-compress, comment images and @mentions, unified workspace chrome, and the black product footer as canonical chrome.

## Highlights

### Product (carried forward + locked)
- **Black product footer** — dark product chrome under light boards (premium light footer experiment rejected; black is canonical)
- **Unified workspace chrome** — header, board surface, and cards share one slate-blue language
- **Comment images** — up to 4 images per comment (D1-backed, auth-served)
- **@mentions & #tags** — autocomplete from project members, in-app notifications
- **Auto image compress** — large screenshots resized/compressed client-side before upload
- **Navigation UX** — save/cancel return to board, login return path, quick-add stays on board
- **Accounts** — register/login, PBKDF2 + JWT, multi-project dashboard
- **Search** — global and project-scoped task search

### Platform
- Cloudflare Pages (Type B direct upload) + Pages Functions
- D1 SQLite (migrations 0001–0005)
- No Express / Railway production path

### Quality & ops
- Full **test campaign** suite (API, Playwright, 100-user sim, UAT catalog)
- Health reports `version: "6.0.0"`
- Marketing README with production screenshots
